# Elm Rails Actioncable Example

## Requirements

* Ruby >= 2.3
* Bundler >= 1.12
* Rails >= 5.0.0.rc1

## How it was initially setup

* `rails new elm-rails-actioncable-example --api --no-skip-action-cable --skip-test --skip-action-mailer`

# Startup

* `bundle`
* `npm i`
* `foreman start -f Procfile.dev`
